# Weather API Web Application

## Description
A responsive web application that fetches and displays real-time weather data using a public REST API.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Fetch API)
- OpenWeatherMap API

## How to Run
1. Open `script.js`
2. Replace `YOUR_API_KEY_HERE` with your OpenWeatherMap API key
3. Open `index.html` in a browser

## Hosting
